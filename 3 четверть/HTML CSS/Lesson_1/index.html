Профессиональный двухподвес

Описание:


Велосипед профессионального уровня, для катания на любых поверхностях. Дорога, горы, грунтовка, камни, лестницы, везде проедет и амортизация прекрасно отработает.

Характеристики товара:


Scott Genius 900 Ultimate (2019) – надежная модель профессионального уровня. Покрышки Maxxis Rekon – лёгкие и износостойкие,
они послужат не один сезон при интенсивном использовании. Карбоновая рама Genius Carbon высокой прочности с эргономичным дизайном.
Система SRAM XX1 DUB Eagle Boost Carbon великолепно передаёт энергию велосипедиста от педалей к звёздочкам.
На модель установлено продвинутое оборудование. За безопасное и своевременно переключение скоростей отвечает задний переключатель SRAM XX1 Eagle.
Дисковые гидравлические тормоза Shimano XTR M9120 надежно схватывают колеса, обеспечивая остановку на любом типе грунта.
Седло Syncros Tofino 1.0 идеально для спортивного катания на высокой скорости.
